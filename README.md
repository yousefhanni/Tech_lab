# Test Product Automation Module

## Overview
This module is a simple test module for demonstrating product automation in Odoo using ORM.

## Features
- **Price Increment**  
  Increases the price of selected products by 10%.

- **VIP Product Detection**  
  Products with an updated price greater than 5000 are collected and logged as VIP products.

- **Status Update**  
  Automatically changes product state from `draft` to `confirmed`.

- **Computed Total Value**  
  Ensures the total stock value is correctly calculated (price Ã— quantity).

## Usage
Select one or more products and click the action button to apply all operations in a single step.

## Purpose
This module is intended for testing, learning, and practicing bulk updates, filtering, logging, and workflow changes in Odoo.
